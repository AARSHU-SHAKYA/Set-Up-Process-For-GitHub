Set Up Process For GitHub

  1. How to create Gmail Account?

    Answer: The steps to create Gmail Account are:

    • Go to the Google Account sign in page.
    • Click Create sccount.
    • Enter suitable name in username field.
    • Create a strong password in the enter password field.
    • Click next. And your google account is ready.

  2. How to create GitHub Account?

    The steps to create GitHub account are as follows:

    • Go to the GitHub Sign In page.
    • CLick Sign in/Create account.
    • In the "username", enter a suitable username.
    • Add your email account.
    • Add your password.
    • Enter the verification code you receive in your email.
    • You have successfully created a GitHub

  3. How to connect VS Code Editor to GitHub account?

    The steps to connect VS Code Editor to GitHub are as follows:

    • Open Visual Code Editor.
    • Go to accounts sign in of VS Code.
    • Download the extension GitHub pull requests and issues.
    • In the accounts, sign in using GitHub account.
    • Open the desired folder in VS Code.
    • DO to the source button, and click initialize repository.
    • Go to terminal, and code git init. (To add the newly created folder in the repository)
    • Now your folder is attached to GitHub.

  4. How to clone/pull repository from GitHub?

    The steps to clone/pull repository from GitHub are as follows:

    • CReate a folder and open in VS Code.
    • Meanwhile, create a repository in your GitHub account.
    • In the "code" section of the repository, copy the url of your repo.
    • Set up your gitbash.(Add global username and global user email)
    • Come back to VS Code, go to the terminal section and enter code got clone (the url you copied) and press enter.
    • You have successfully cloned you rrepository to the VS Code.

  5. How to push repository/folder to GitHub?

    The steps to push repository in GitHb are as follows:

    • Once, VS Code and github are made in sync, any changes to the file must be scanned by the git.
    • For that, three steps are to be performed "add,commit adnd push".
    • After the changes are addedusing code"git add (file name)",git commit is to be done by code "git commit -m "Suitable phrase to justify the content", and finally git push is done.
    • Git push command is given by code "git push origin main". Now, your file is being uploaded into your github repository.

  6. How to publish the repo as live site?

    THe steps to publish the repo as live site are as follows:

    • Open your GitHub repository.
    • Under settings, go to pages.
    • Choose right options like deploy from a branch,custom domain and then click save.
    • Now, your repo should be hosted live.
